<task type="auto">
  <name>Create Arm assembly class</name>
  <files>projects/tile-platform/lib/arms/arm.py</files>
  <action>
Create Arm class representing complete arm assembly:
1. Define Arm class with:
   - segments: List[ArmSegmentConfig]
   - joints: JointChain
   - state: ArmState
   - base_position: Tuple[float, float, float]
   - end_effector_position: Tuple[float, float, float]

2. Implement methods:
   - calculate_end_position(joint_angles: List[float]) -> Tuple[float, float, float]
     * Forward kinematics through all joints
     * Returns world position of end effector

   - solve_for_target(target_pos: Tuple[float, float, float]) -> List[float]
     * Inverse kinematics to find joint angles
     * Returns angles that place end effector at target

   - update(dt: float, target_pos: Tuple[float, float, float])
     * Performs physics simulation step
     * Updates joint angles
     * Updates end effector position

   - get_bounds() -> Tuple[float, float, float, float, float, float]
     * Returns bounding box of arm in current configuration

3. Add factory methods:
   - create_standard_arm(base_pos: Tuple[float, float, float]) -> Arm
     * Creates typical 3-segment arm
     * Uses standard joint configurations

   - create_long_arm(base_pos: Tuple[float, float, float]) -> Arm
     * Creates extended 4-segment arm for longer reach
  </action>
  <verify>python -c projects/tile-platform/lib/arms/arm.py</verify>
  <done>Arm class with forward/inverse kinematics, factory methods</done>
</task>

<task type="auto">
  <name>Create ArmController</name>
  <files>projects/tile-platform/lib/arms/controller.py</files>
  <action>
Create ArmController for managing multiple arms:
1. Define ArmController class with:
   - arms: List[Arm]
   - physics_simulator: PhysicsSimulator
   - target_assignments: Dict[int, Tuple[float, float, float]]

2. Implement methods:
   - add_arm(arm: Arm) -> int
     * Adds arm to controller, returns arm index

   - remove_arm(arm_index: int) -> bool
     * Removes arm from controller

   - assign_target(arm_index: int, target_pos: Tuple[float, float, float])
     * Assigns target position to specific arm

   - update_all(dt: float)
     * Updates all arms towards their targets
     * Handles physics simulation for all arms

   - get_arm_positions() -> Dict[int, Tuple[float, float, float]]
     * Returns current end effector positions for all arms

   - get_available_arm(target_pos: Tuple[float, float, float]) -> Optional[int]
     * Finds arm closest to target that can reach it
     * Returns arm index or None if no arm available

3. Add collision avoidance:
   - check_arm_collisions() -> List[Tuple[int, int]]
     * Returns pairs of arms that are colliding
     * Used for visual feedback
  </action>
  <verify>python -c projects/tile-platform/lib/arms/controller.py</verify>
