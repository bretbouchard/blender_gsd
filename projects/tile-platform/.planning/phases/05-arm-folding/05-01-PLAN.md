<task type="auto">
  <name>Create folding animator</name>
  <files>projects/tile-platform/lib/folding/animator.py</files>
  <action>
Create FoldingAnimator for smooth transitions:
1. Define FoldingAnimator class with:
   - arms: List[Arm]
   - folding_configs: Dict[int, FoldingConfig]
   - transition_easing: str (default "ease_in_out")

2. Implement methods:
   - fold_arm(arm_index: int) -> None
     * Initiates folding animation
     * Updates FoldingPose to FOLDING
     * Starts timer

   - deploy_arm(arm_index: int) -> None
     * Initiates deployment animation
     * Updates FoldingPose to DEPLOYING

   - update(dt: float) -> Dict[int, FoldingPose]
     * Advances all animations
     * Returns updated poses
     * Handles easing functions

   - get_folded_positions() -> List[int]
     * Returns indices of arms in FOLDED state

   - get_deployed_positions() -> List[int]
     * Returns indices of arms in DEPLOYED state

3. Easing functions:
   - Linear: t
   - Ease-in-out: t^2 * (3 - 2*t)
   - Smooth: t^3 * (t * (6*t - 15) + 10)
  </action>
  <verify>python -c projects/tile-platform/lib/folding/animator.py</verify>
