# Phase 6.3 Plan 1: Backdrop Config Extension and Preset Loaders

## Summary

Extended BackdropConfig dataclass with properties for infinite curves, gradients, HDRI, and mesh backdrops. Added backdrop preset loader functions following the existing pattern used by the lighting system.

## One-liner

Extended BackdropConfig with curve/gradient/HDRI/mesh properties and added 6 backdrop preset loader functions.

## Changes

### Modified Files

| File | Changes |
|------|---------|
| lib/cinematic/types.py | Extended BackdropConfig with 7 new properties |
| lib/cinematic/preset_loader.py | Added BACKDROP_CONFIG_ROOT constant and 6 loader functions |

### New Properties in BackdropConfig

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| curve_height | float | 3.0 | Height of vertical wall for infinite curves (meters) |
| curve_segments | int | 32 | Resolution of curve geometry |
| gradient_type | str | "linear" | Gradient type (linear, radial, angular) |
| gradient_stops | List[Dict] | [] | Gradient color stops with position and color |
| hdri_preset | str | "" | Name of HDRI preset for HDRI backdrops |
| mesh_file | str | "" | Path to mesh file for mesh backdrops |
| mesh_scale | float | 1.0 | Scale factor for mesh environments |

### New Preset Loader Functions

| Function | Config File | Key Access |
|----------|-------------|------------|
| get_infinite_curve_preset(name) | infinite_curves.yaml | curves |
| get_gradient_preset(name) | gradients.yaml | gradients |
| get_environment_preset(name) | environments.yaml | environments |
| list_infinite_curve_presets() | infinite_curves.yaml | curves |
| list_gradient_presets() | gradients.yaml | gradients |
| list_environment_presets() | environments.yaml | environments |
