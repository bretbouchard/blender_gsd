## Supported Gel Types
- **CTO (Color Temperature Orange)**: cto_full, cto_half, cto_quarter, cto_eighth
- **CTB (Color Temperature Blue)**: ctb_full, ctb_half, ctb_quarter, ctb_eighth
- **Diffusion**: diffusion_full, diffusion_half, diffusion_quarter
- **Creative Colors**: red, blue, amber, magenta, green, cyan
- **Combined**: cto_diffusion, ctb_diffusion

## Features
- Tanner Helland algorithm for Kelvin to RGB conversion
- Blender 4.0+ native temperature support with fallback
- Gel combining with color multiplication
- Preset-based gel loading
