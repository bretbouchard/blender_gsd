## Changes Made

### 1. Extended LightConfig (types.py)
Added light-type-specific properties:
- `shape: str = "RECTANGLE"` - Area light shape (SQUARE, RECTANGLE, DISK, ELLIPSE)
- `size: float = 1.0` - Width/radius for area lights
- `size_y: float = 1.0` - Height for RECTANGLE/ELLIPSE
- `spread: float = 1.047` - Spread angle in radians (60° default)
- `spot_size: float = 0.785` - Spot cone angle in radians (45° default)
- `spot_blend: float = 0.5` - Spot edge softness (0-1)
- `shadow_soft_size: float = 0.1` - Shadow softness radius
- `use_shadow: bool = True` - Enable/disable shadows
- `use_temperature: bool = False` - Use Kelvin color temperature
- `temperature: float = 6500.0` - Color temperature in Kelvin

### 2. New Dataclasses (types.py)
- **GelConfig** - Light gel/color filter configuration
  - name, color, temperature_shift, softness, transmission, combines
  
- **HDRIConfig** - HDRI environment lighting configuration
  - name, file, exposure, rotation, background_visible, saturation
  
- **LightRigConfig** - Lighting rig preset configuration
  - name, description, extends, lights, hdri

### 4. Lighting Preset Loaders (preset_loader.py)
Added `LIGHTING_CONFIG_ROOT = Path("configs/cinematic/lighting")` constant.

New functions:
- `get_lighting_rig_preset(name)` - Load lighting rig preset
- `get_gel_preset(name)` - Load gel/color filter preset
- `get_hdri_preset(name)` - Load HDRI preset
- `list_lighting_rig_presets()` - List available lighting rig presets
- `list_gel_presets()` - List available gel presets
- `list_hdri_presets()` - List available HDRI presets
