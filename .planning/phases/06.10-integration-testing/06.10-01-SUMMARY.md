## Key Changes

### IntegrationConfig
- Connects cinematic rendering with control surface system
- Controls material sync, animation sync, and auto-framing
- Default render modes: hero, catalog, turntable

### TestConfig
- End-to-end rendering test scenario configuration
- Shot configuration with expected render passes
- Validation checks with configurable tolerance

### PerformanceConfig
- Performance targets for preview (10s) and production (120s/frame) renders
- Orbit animation target (50ms) and shot assembly target (100ms)
- Memory limit (4GB) and GPU utilization (80%) targets
- Optional profiling with output path

### BenchmarkResult
- Stores benchmark measurements (duration, memory, GPU utilization)
- Tracks frames rendered and passes generated
- ISO timestamp and pass/fail status
