**Rig Types with Constraints**:
| Rig Type | Constraints Applied |
|----------|---------------------|
| tripod | TRACK_TO + lock_location |
| tripod_orbit | TRACK_TO only |
| dolly | TRACK_TO + lock Y,Z |
| dolly_curved | TRACK_TO + lock Z |
| crane | TRACK_TO + LIMIT_ROTATION |
| steadicam | COPY_LOCATION + TRACK_TO with smoothing |
| drone | TRACK_TO with low influence |

**Multi-Camera Layout Types**:
- `grid`: NxN grid arrangement
- `horizontal`: Horizontal line
- `vertical`: Vertical stack
- `circle`: Circle around origin
- `arc`: 180-degree arc
- `custom`: Use explicit positions

**Multi-Camera Composite**:
- Creates Render Layers node for each camera
- Uses Translate nodes to position in grid
- Combines with Mix nodes (ADD blend)
- Scales output resolution by grid dimensions
