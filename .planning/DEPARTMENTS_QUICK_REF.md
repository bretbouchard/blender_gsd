# Film Production Departments Quick Reference

## 15 Departments + Retro Graphics

### 1. DEVELOPMENT
Story, script, creative vision
- **Producer** → Production Orchestrator
- **Writer** → Script Parser
- **Creative Director** → Style Preset System

### 2. PRE-VISUALIZATION
Plan shots before production
- **Previs Supervisor** → Previs Orchestrator
- **Story Artist** → Storyboard Generator
- **Animatic Editor** → Animatic Builder

### 3. ART DEPARTMENT
Visual design of everything
- **Production Designer** → Design Director
- **Art Director** → Asset Coordinator
- **Set Designer** → Set Builder
- **Prop Master** → Prop Library
- **Concept Artist** → Concept Generator

### 4. CAMERA DEPARTMENT ✅ (BUILT)
Capture the image
- **Director of Photography** → Shot Config System
- **Camera Operator** → Camera Controller
- **1st AC** → Focus Puller
- **DIT** → Render Settings

### 5. GRIP DEPARTMENT
Camera support and movement
- **Key Grip** → Grip Supervisor
- **Dolly Grip** → Dolly Controller
- **Rigging Grip** → Rig Builder

### 6. ELECTRICAL DEPARTMENT ✅ (BUILT)
Power and lighting power
- **Gaffer** → Lighting Director
- **Best Boy Electric** → Equipment Manager
- **Electrician** → Light Placer

### 7. LIGHTING DEPARTMENT ✅ (BUILT)
Shape light for mood
- **Gaffer** → Light Director
- **Lighting Designer** → Light Plot Generator
- **Board Operator** → Light Controller

### 8. WARDROBE DEPARTMENT
Character costumes
- **Costume Designer** → Wardrobe Director
- **Costume Maker** → Costume Generator
- **Dresser** → Dressing System

### 9. HAIR & MAKEUP DEPARTMENT
Character appearance
- **Makeup Designer** → Makeup Director
- **Hair Stylist** → Hair Styler
- **Prosthetics Artist** → Prosthetic Builder

### 10. LOCATION DEPARTMENT
Find and manage locations
- **Location Manager** → Location Director
- **Location Scout** → Location Finder

### 11. PRODUCTION DEPARTMENT
Schedule and coordinate
- **Line Producer** → Production Manager
- **1st AD** → Schedule Manager
- **Script Supervisor** → Continuity Tracker

### 12. SOUND DEPARTMENT
Production sound
- **Production Sound Mixer** → Sound Director
- **Boom Operator** → Boom Positioner

### 13. VFX DEPARTMENT
Visual effects
- **VFX Supervisor** → VFX Director
- **Compositor** → Compositor
- **CG Supervisor** → CG Director
- **FX Artist** → FX Simulator

### 14. EDITORIAL DEPARTMENT
Assemble final cut
- **Editor** → Edit System
- **Colorist** → Color System

### 15. POST-PRODUCTION SOUND
Final soundtrack
- **Supervising Sound Editor** → Sound Director
- **Sound Designer** → Sound Designer
- **Re-Recording Mixer** → Mixer
- **Composer** → Score Generator

### 16. RETRO GRAPHICS DEPARTMENT (NEW)
Hollywood to Double Dragon
- **Style Director** → Style Orchestrator
- **Pixel Artist** → Pixel Converter
- **Quantizer** → Color Quantizer
- **Ditherer** → Dither Engine
- **Isometric Designer** → Iso Converter
- **Sprite Animator** → Sprite System

