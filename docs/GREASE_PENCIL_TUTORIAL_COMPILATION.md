## Key Techniques Across All Tutorials

### Animation Methods
- **Frame-by-frame** - Traditional 2D animation
- **Cut-out** - Rigged 2D characters
- **Hybrid** - 2D elements in 3D scenes
- **Rotoscoping** - Tracing video reference

### Rigging Approaches
- **Bone rigging** - Armature-based deformation
- **Vertex groups** - Weight painting strokes
- **Drivers** - Automated secondary motion
- **Parenting** - Simple hierarchy animation

### Material Systems
- **Stroke shaders** - Line appearance control
- **Fill materials** - Color and texture for fills
- **Hybrid shaders** - 2D/3D integration
- **Custom NPR** - Anime and stylized looks

### Integration Techniques
- **Drawing on 3D** - Strokes on mesh surfaces
- **2.5D effects** - Hybrid visual styles
- **V-storyboarding** - Visual planning
- **Compositing** - Post-processing 2D elements
